Anotações

Os arquivos do Java nada mais são do que um arquivo de texto.

Uma IDE é um ambiente de desenvolvimento integrado e nos fornece várias ferramentas para auxiliar no desenvolvimento, como:

aceleração de escrita com auto-complete ou IntelliSense

Formatação de palavras e blocos de códigos

Análise de erro de sintaxe

Compilação de programa

Depuração (acompanhamento) de execução de programa.

> Cada IDE tem recursos comuns dos ou plugins específicos para a agilidade de desenvolvimento



Principais IDEs:

Eclipse: utilizada mais em ambientes corporativos com interface bem interativa

NetBeans: utilizada mais em ambientes acadêmicos como versão de menu e assistentes em português.

IntelliJ: Oferece muitos recursos de inteligência para agilidade no nosso desenvolvimento, com o uso da IDE, ela passa a dar sugestões de soluções bem assertivas. (Nem todos os recursos estão disponíveis de forma gratuita)

VS Code: queridinho das IDEs pelos desenvolvedores que já atuam com NodeJs ou plataformas de front-end. Serve para Java, JavaScript, NodeJS.

Comparação de atalhos:

| Comando                             | Eclipse                    | IntelliJ         |
| ----------------------------------- | -------------------------- | ---------------- |
| Cria o método principal main        | main, ctrl + espaço, enter | psvm             |

| Método de impressão no Console      | sysout, ctrl + espaço      | sout             |
| Renomeando variável ou método       | alt+shift+R                | shift + F6       |
| Localizar a utilização de um método | ctrl+shift+G               | alt + F7         |
| Entrar na implementação do método   | ctrl+click                 | ctrl+click       |
